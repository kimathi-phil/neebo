<p align="center">A simple Discord bot written in Rust</p>

## Features
Commands currently supported by Neebo:

### Help
Learn what Neebo can do for you
```
?help
```

### Ban
Ban a user
```
?ban {user}
```
### Kick
Kick a user
```
?kick {user}
```

### Code of Conduct
Displays the code of conduct in the specified channel
```
?CoC {channel}
```
